Bootstrap Notes 

- Bootstrap 4 will style the HTML <mark> element with a yellow background color and some padding
- Bootstrap 4 will style the HTML <abbr> element with a dotted border bottom
- A jumbotron class indicates a big grey box for calling extra attention to some special content or information.



    

Bootstrap's grid system is built with flexbox and allows up to 12 columns across the page.

align-items is always for cross axis. not for main axis .
Cross axis can be vertical or horizontal depending whether items are row or colomn
If items are in row order, main axis is vertical line. 
If items are in column order, main axis is horizontal line. 

justify-content is always for main axis. 
Main axis can be vertical or horizontal depending whether items are row or colomn
